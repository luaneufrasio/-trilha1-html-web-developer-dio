Ferramentas Utilizadas

Para o uso do HTML se usa duas ferramentas simples, um editor de texto seja ele bloco de notas, notepad e o principal e mais usado, VsCode e o navegador seja ele chrome, mozilla firefox, Opera, Microsoft edge, Safari etc...

Inspetor de Elementos

O inspetor de elementos é utilizado no navegador para visualizar as linhas de código do html e saber em qual posição se trata cada uma.

Estrutura básica do HTML

Inicializando com HTML criamos uma pasta que por padrão e para melhor definir a página principal chamamos de index.html e todo html consiste em head que é o cabeçalho da página e temos o body que define o corpo da página e também o title que altera o nome da página exibido na aba do navegador.

Falando sobre Tags

As tags são os elementos que compõe a principal esfera do HTML citando algumas delas

  • HTML indicando que aquilo é uma página Html
  • Head é "o que" pré carrega antes de exibir para o usuário nas demais tags
  • Body indica o corpo da página
Existem inúmeras tags no html e a escrita é exibida por < i > < /i > contendo simbolos de mais (>) ou de menos (<) e fechando com outra identica porém com uma ( / ) indicando o final daquela tag essa tag I por exemplo transforma o que estiver escrito dentro dela em itálico exemplo Texto em Itálico.

Atributos das tags

No HTML existem inúmeros atributos para diversas tags, mas citando os mais utilizados vamos começar pelo Id. O id é um atributo de "Identificação" ele denomina um caminho para aquela tag com o nome inserido em seu conteúdo exemplo: id="qualquerNome" ;

Textos

Para exibição de texto no html temos as tags de h1 a h6 alterando o tamanho da fonte conforme muda-se o valor do H Temos também a tag P que simboliza parágrafo no qual pula uma linha e inicia-se um novo parágrafo, a tag blockquote adiciona uma citação centralizando o texto abaixo do anterior, temos também a tag strong que transforma aquele texto ou palavra em negrito

Listas Ordenadas e não ordenadas

No HTML temos dois tipos de listas, as ordenadas e as não ordenadas. As ordenadas que no ingles "Ordered list" utilizamos a tag ol que exibe uma lista de forma enumerada exemplo:

  1. "Ordered List"
  2. "Ordered List"
Já listas não ordenadas ou "list unordered" elas possuem marcações no começo da escrita de cada linha e assim como o nome diz ela não segue uma ordem, exemplo:
  • "Ordered List"
  • "Ordered List"

Links

Fazer conexão a outros sites ou "linkar" criar links pra isso no HTML usammos a tag a de âncora, porém ao clicar no link ele abre na mesma aba e então temos alguns atributos adicionais caso seja necessario essa mudança uma delas é o atributo Target que abre algumas opções como "_blank" que abre o link em uma nova guia(aba) e "_self"que como indica a tradução ele irá abrir o link nele mesmo ou seja no proprio site, Também há como adicionar o atributo Title assim exibe uma mensagem ao manter o mouse acima do link.
